Simple ajax login widget for your WordPress! show avatar, profile button, register and forgot password forms.

== Description ==

This plugin allow you to display login widget in your WordPress. It’s useful for sites that would like to avoid the normal WordPress login pages. Login uses AJAX and jQuery effects.

Features:

1. Login Form.
2. Register Form.
3. Forgot Password Form.
4. Error handling without refreshing your screen.
5. Optional ReCaptcha verification for login and register forms.
6. Control forgotten password request.
7. Work with forced SSL logins.
8. Specific option to show link to profile page and dashboard.
9. Customizable redirect URL after log in or log out.
10. Option to show or hide avatar and change it size.
11. It's responsive!

== Installation ==

1. Upload ajax-wp-login folder to wp-content/plugins.
2. Click "Activate" in the WordPress plugins menu.
1. Place "Login" widget in your sidebar.
